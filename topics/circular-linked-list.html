Circular Linked List

Phase 6 Medium

📖 What is a Circular Linked List?

A Circular Linked List is a linked list where the last node points back to the first node instead of null. It forms a continuous loop — there's no "end."

🌍 Real-Life Analogy

Think of people sitting in a circle playing a game (like musical chairs or hot potato). After the last person, you're back to the first — it's a continuous loop! A regular linked list is like people standing in a line with a clear start and end. A circular list is the circle — no end, just keep going around.

⚙️ How It Works

Circular Linked List Structure
Regular: [10] → [20] → [30] → null Circular: [10] → [20] → [30] ─┐ ↑ | └────────────────────┘ (last node points to head)
JavaScript
class Node {
    constructor(data) {
        this.data = data;
        this.next = null;
    }
}

class CircularLinkedList {
    constructor() {
        this.head = null;
        this.size = 0;
    }
    
    // Insert at end — O(n) or O(1) with tail
    insert(data) {
        const node = new Node(data);
        if (!this.head) {
            this.head = node;
            node.next = node;  // Points to itself!
        } else {
            let curr = this.head;
            while (curr.next !== this.head) curr = curr.next;
            curr.next = node;
            node.next = this.head;  // Complete the circle
        }
        this.size++;
    }
    
    // Delete a value — O(n)
    delete(data) {
        if (!this.head) return;
        
        // If head is the target
        if (this.head.data === data) {
            if (this.head.next === this.head) {
                this.head = null;  // Only one node
            } else {
                let last = this.head;
                while (last.next !== this.head) last = last.next;
                this.head = this.head.next;
                last.next = this.head;
            }
            this.size--;
            return;
        }
        
        let curr = this.head;
        while (curr.next !== this.head && curr.next.data !== data) {
            curr = curr.next;
        }
        if (curr.next.data === data) {
            curr.next = curr.next.next;
            this.size--;
        }
    }
    
    // Traverse — must stop when we loop back to head
    print() {
        if (!this.head) return '(empty)';
        let curr = this.head;
        const result = [];
        do {
            result.push(curr.data);
            curr = curr.next;
        } while (curr !== this.head);
        return result.join(' → ') + ' → (back to head)';
    }
}

// Josephus Problem: n people in circle, every k-th eliminated
function josephus(n, k) {
    let people = Array.from({length: n}, (_, i) => i + 1);
    let idx = 0;
    while (people.length > 1) {
        idx = (idx + k - 1) % people.length;
        people.splice(idx, 1);
    }
    return people[0];  // Survivor
}

⚠️ Common Mistakes

❌ Infinite loop during traversal
// ❌ Never-ending loop!
let curr = head;
while (curr) {           // Always true in circular list!
    curr = curr.next;
}

// ✅ Use do-while, stop when back at head
let curr = head;
do {
    // process curr
    curr = curr.next;
} while (curr !== head);
❌ Forgetting single-node edge case

When there's only one node, it points to itself. Deleting it means setting head = null, not just updating pointers.

💼 Interview Perspective

  • Josephus Problem is the classic circular list interview question.
  • Used in round-robin scheduling (CPU time slicing) and circular buffers.
  • Traversal must use do-while or track the starting node to avoid infinite loops.
  • Maintaining a tail pointer makes insertion O(1) at both head and tail.
  • Can be singly circular or doubly circular (with prev pointers too).

📝 Quick Revision Notes

  • Last node's next → head. No null at the end.
  • Traversal: use do-while, stop when curr === head.
  • Used in: Josephus problem, round-robin scheduling, circular buffers.
  • Single node: points to itself. Empty list: head = null.

🃏 Flash Cards

How do you detect if a linked list is circular?
Use the fast & slow pointer technique. If they meet, it's circular. OR simply traverse and check if any node's next equals head. In a properly constructed circular list, the last node's next IS head.
What is the Josephus Problem?
N people stand in a circle. Every k-th person is eliminated. Find the last survivor. Classic circular list application. Can be solved with simulation O(nk) or math formula O(n): J(n,k) = (J(n-1,k) + k) % n.
