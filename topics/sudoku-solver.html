Sudoku Solver

Phase 13 Hard

📖 What is Sudoku Solving?

A Sudoku Solver fills a 9×9 grid so that each row, column, and 3×3 sub-box contains digits 1-9 exactly once. This is a constraint satisfaction problem solved with backtracking — try a number, check constraints, backtrack if invalid.

💡 Key Insight

For each empty cell, try digits 1-9. Check validity against three constraints: row, column, and 3×3 box. If valid, place it and move to the next empty cell. If no digit works, backtrack. The 3×3 box index is 3*Math.floor(row/3) + Math.floor(col/3).

🌍 Real-Life Analogy

🧩 Crossword Puzzle

Solving Sudoku is like filling in a crossword: you try a word (number), check if it fits the constraints (row/column/box), and if you get stuck, you erase and try a different word. The constraints narrow down possibilities at each step.

⚙️ Sudoku Solver Implementation

JavaScript — Sudoku Solver (LC 37)
function solveSudoku(board) {
    solve(board);
}

function solve(board) {
    for (let row = 0; row < 9; row++) {
        for (let col = 0; col < 9; col++) {
            if (board[row][col] !== '.') continue;

            for (let num = 1; num <= 9; num++) {
                const char = String(num);
                if (isValid(board, row, col, char)) {
                    board[row][col] = char;  // Place number

                    if (solve(board)) return true; // Solved!

                    board[row][col] = '.';   // Backtrack
                }
            }
            return false; // No valid number for this cell → backtrack
        }
    }
    return true; // All cells filled
}

function isValid(board, row, col, char) {
    for (let i = 0; i < 9; i++) {
        // Check row
        if (board[row][i] === char) return false;
        // Check column
        if (board[i][col] === char) return false;
        // Check 3x3 box
        const boxRow = 3 * Math.floor(row / 3) + Math.floor(i / 3);
        const boxCol = 3 * Math.floor(col / 3) + (i % 3);
        if (board[boxRow][boxCol] === char) return false;
    }
    return true;
}

// Optimized version using Sets for O(1) lookup
function solveSudokuOptimized(board) {
    const rows = Array.from({length: 9}, () => new Set());
    const cols = Array.from({length: 9}, () => new Set());
    const boxes = Array.from({length: 9}, () => new Set());
    const empty = [];

    // Initialize constraints from given numbers
    for (let r = 0; r < 9; r++) {
        for (let c = 0; c < 9; c++) {
            if (board[r][c] !== '.') {
                const num = board[r][c];
                rows[r].add(num);
                cols[c].add(num);
                boxes[3 * Math.floor(r/3) + Math.floor(c/3)].add(num);
            } else {
                empty.push([r, c]);
            }
        }
    }

    function backtrack(idx) {
        if (idx === empty.length) return true;
        const [r, c] = empty[idx];
        const box = 3 * Math.floor(r/3) + Math.floor(c/3);

        for (let num = 1; num <= 9; num++) {
            const ch = String(num);
            if (rows[r].has(ch) || cols[c].has(ch) || boxes[box].has(ch)) continue;

            board[r][c] = ch;
            rows[r].add(ch); cols[c].add(ch); boxes[box].add(ch);

            if (backtrack(idx + 1)) return true;

            board[r][c] = '.';
            rows[r].delete(ch); cols[c].delete(ch); boxes[box].delete(ch);
        }
        return false;
    }

    backtrack(0);
}

⚠️ Common Mistakes

❌ Wrong 3×3 box index calculation

Box index = 3 * Math.floor(row/3) + Math.floor(col/3). Getting this wrong means the box constraint check fails silently.

❌ Not returning false when no digit works

After trying all 9 digits for an empty cell without success, you MUST return false to trigger backtracking. Missing this causes the algorithm to skip cells.

📝 Quick Revision Notes

  • Sudoku: fill 9×9 grid with 1-9. Each row, col, 3×3 box has unique digits.
  • Backtracking: try 1-9 for each empty cell. Check row + col + box. Backtrack if stuck.
  • Box index: 3*(row/3) + (col/3) (integer division).
  • Optimize with Sets for O(1) constraint checking.

🃏 Flash Cards

What are the three constraints in Sudoku?
1. Each row contains 1-9 exactly once. 2. Each column contains 1-9 exactly once. 3. Each 3×3 sub-box contains 1-9 exactly once. A number placement must satisfy all three simultaneously.
How do you calculate the 3×3 box index from (row, col)?
boxIndex = 3 * Math.floor(row / 3) + Math.floor(col / 3). This maps (0-2, 0-2)→box 0, (0-2, 3-5)→box 1, etc. There are 9 boxes numbered 0-8.
