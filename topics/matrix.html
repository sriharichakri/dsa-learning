Matrix (2D Arrays)

Phase 2 Medium

📖 What is a Matrix?

A matrix (2D array) is an array of arrays — data organized in rows and columns. You access elements using two indices: matrix[row][col].

Matrices are used to represent grids, images, game boards, graphs (adjacency matrix), spreadsheets, and much more.

💡 Key Insight

Matrix problems often require specific traversal patterns: row-by-row, column-by-column, diagonal, spiral, or boundary traversal. Knowing these patterns is essential for interviews.

🌍 Real-Life Analogy

♟️ Chessboard / Spreadsheet Grid

A chessboard is an 8×8 matrix. Each square is identified by its row and column (e.g., D4). A spreadsheet works the same way — cell B3 is row 3, column B. Matrices give us the power to store and manipulate 2D data efficiently.

  • Image pixels: Each pixel at (row, col) stores RGB values
  • Seating chart: Rows and seats in a theater
  • Game board: Tic-tac-toe, Sudoku, Minesweeper — all matrices
  • Map/grid: City blocks, terrain maps with elevation data

Spiral Order Traversal

JavaScript — Spiral Matrix
function spiralOrder(matrix) {
    const result = [];
    let top = 0, bottom = matrix.length - 1;
    let left = 0, right = matrix[0].length - 1;
    
    while (top <= bottom && left <= right) {
        // Traverse right
        for (let c = left; c <= right; c++) result.push(matrix[top][c]);
        top++;
        // Traverse down
        for (let r = top; r <= bottom; r++) result.push(matrix[r][right]);
        right--;
        // Traverse left
        if (top <= bottom) {
            for (let c = right; c >= left; c--) result.push(matrix[bottom][c]);
            bottom--;
        }
        // Traverse up
        if (left <= right) {
            for (let r = bottom; r >= top; r--) result.push(matrix[r][left]);
            left++;
        }
    }
    return result;
}

🔄 Rotate Image (90° Clockwise)

Trick: Transpose the matrix, then reverse each row.

JavaScript — Rotate 90° Clockwise
function rotate(matrix) {
    const n = matrix.length;
    
    // Step 1: Transpose (swap rows and columns)
    for (let i = 0; i < n; i++) {
        for (let j = i + 1; j < n; j++) {
            [matrix[i][j], matrix[j][i]] = [matrix[j][i], matrix[i][j]];
        }
    }
    
    // Step 2: Reverse each row
    for (let i = 0; i < n; i++) {
        matrix[i].reverse();
    }
}

🔍 Search a 2D Matrix

JavaScript — Staircase Search O(m+n)
// Matrix where each row and column is sorted
function searchMatrix(matrix, target) {
    let row = 0;
    let col = matrix[0].length - 1;  // Start at top-right corner
    
    while (row < matrix.length && col >= 0) {
        if (matrix[row][col] === target) return true;
        if (matrix[row][col] > target) col--;   // Too big → go left
        else row++;                               // Too small → go down
    }
    return false;
}
// Starts at top-right. Each step eliminates a row or column.

🔍 Dry Run: Set Matrix Zeroes

JavaScript
function setZeroes(matrix) {
    const rows = matrix.length, cols = matrix[0].length;
    let firstRowZero = false, firstColZero = false;
    
    // Check if first row/col need zeroing
    for (let c = 0; c < cols; c++) if (matrix[0][c] === 0) firstRowZero = true;
    for (let r = 0; r < rows; r++) if (matrix[r][0] === 0) firstColZero = true;
    
    // Use first row/col as markers
    for (let r = 1; r < rows; r++) {
        for (let c = 1; c < cols; c++) {
            if (matrix[r][c] === 0) {
                matrix[r][0] = 0;
                matrix[0][c] = 0;
            }
        }
    }
    
    // Zero based on markers
    for (let r = 1; r < rows; r++)
        for (let c = 1; c < cols; c++)
            if (matrix[r][0] === 0 || matrix[0][c] === 0)
                matrix[r][c] = 0;
    
    if (firstRowZero) for (let c = 0; c < cols; c++) matrix[0][c] = 0;
    if (firstColZero) for (let r = 0; r < rows; r++) matrix[r][0] = 0;
}

⚠️ Common Mistakes

❌ Confusing Row and Column Indices

matrix[row][col] — row comes first! For an m×n matrix: m rows, n columns. matrix.length = rows, matrix[0].length = cols.

❌ Out of Bounds in Direction Traversal

When exploring 4 or 8 directions, always check bounds: 0 <= newRow < rows && 0 <= newCol < cols before accessing.

❌ Wrong Spiral Boundary Checks

After moving top++ and right--, you must check top <= bottom before traversing left, and left <= right before traversing up to avoid duplicating elements.

💼 Interview Tips

  • Direction array pattern: const dirs = [[0,1],[0,-1],[1,0],[-1,0]] for 4-directional traversal
  • Rotate = Transpose + Reverse: Memorize this trick. 90° CW = transpose + reverse rows. 90° CCW = transpose + reverse columns.
  • In-place markers: Use first row/column as markers to avoid O(m+n) extra space (Set Matrix Zeroes)
  • Layer-by-layer: For spiral/rotate, think in layers from outside to inside
  • Flatten trick: Treat m×n matrix as 1D array of size m*n. Index (r,c) = r*cols + c. Row = idx/cols, Col = idx%cols.

🏗️ Where Matrices Are Used

  • Image processing: Every image is a matrix of pixel values
  • Game boards: Chess, Sudoku, Minesweeper — all grid-based
  • Spreadsheets: Excel/Google Sheets are essentially matrices
  • Machine learning: Data stored as matrices, operations use matrix multiplication
  • Graph representation: Adjacency matrix for dense graphs

📝 Quick Revision Notes

  • Matrix = 2D array. Access: matrix[row][col]. Rows = matrix.length, Cols = matrix[0].length
  • Rotate 90° CW: transpose + reverse each row
  • Spiral: use top/bottom/left/right boundaries, shrink after each direction
  • Sorted matrix search: start at top-right, go left if too big, down if too small → O(m+n)
  • Direction array: [[0,1],[0,-1],[1,0],[-1,0]] for 4-directional movement

🃏 Flash Cards

How do you rotate a matrix 90° clockwise in-place?
Two steps: 1) Transpose the matrix (swap matrix[i][j] and matrix[j][i]). 2) Reverse each row. Result: 90° clockwise rotation in O(n²) time, O(1) space.
How does the staircase search work on a sorted matrix?
Start at top-right corner. If current > target, go left (smaller). If current < target, go down (larger). Each step eliminates a row or column. Time: O(m + n).
How to convert 2D matrix index to 1D and back?
To 1D: index = row * cols + col. To 2D: row = index / cols (integer division), col = index % cols. Useful for treating a matrix as a flat array.
