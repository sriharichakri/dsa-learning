Longest Common Subsequence (LCS)

Phase 12 Medium

📖 What is LCS?

The Longest Common Subsequence problem finds the longest sequence of characters that appears in both strings, in the same relative order, but not necessarily contiguous.

For strings "ABCDE" and "ACE", the LCS is "ACE" (length 3). Note: subsequence ≠ substring. A subsequence doesn't need to be contiguous.

💡 Key Insight

LCS is the classic 2D DP problem. Compare characters: if they match, extend the LCS by 1. If not, take the best of skipping either character. The recurrence: dp[i][j] = text1[i-1] === text2[j-1] ? dp[i-1][j-1]+1 : max(dp[i-1][j], dp[i][j-1]).

🌍 Real-Life Analogy

🧬 DNA Sequence Matching

Scientists compare DNA sequences to find common patterns. Two DNA strands might not be identical, but they share a long common subsequence indicating evolutionary relationship. LCS finds the longest such shared pattern — crucial in bioinformatics.

⚙️ LCS Implementation

DP Table for LCS("ABCDE", "ACE")
"" A C E "" [ 0 0 0 0 ] A [ 0 1 1 1 ] ← A matches A → dp[1][1]=1 B [ 0 1 1 1 ] ← B doesn't match → take max C [ 0 1 2 2 ] ← C matches C → dp[3][2]=2 D [ 0 1 2 2 ] ← D doesn't match → take max E [ 0 1 2 3 ] ← E matches E → dp[5][3]=3 LCS length = dp[5][3] = 3 ("ACE")
JavaScript — LCS
function longestCommonSubsequence(text1, text2) {
    const m = text1.length, n = text2.length;
    const dp = Array.from({length: m + 1}, () => new Array(n + 1).fill(0));

    for (let i = 1; i <= m; i++) {
        for (let j = 1; j <= n; j++) {
            if (text1[i - 1] === text2[j - 1]) {
                dp[i][j] = dp[i - 1][j - 1] + 1; // Characters match
            } else {
                dp[i][j] = Math.max(dp[i - 1][j], dp[i][j - 1]); // Take best
            }
        }
    }
    return dp[m][n];
}

// Space-optimized: O(n) space
function lcsOptimized(text1, text2) {
    const n = text2.length;
    let prev = new Array(n + 1).fill(0);

    for (let i = 1; i <= text1.length; i++) {
        const curr = new Array(n + 1).fill(0);
        for (let j = 1; j <= n; j++) {
            if (text1[i - 1] === text2[j - 1]) {
                curr[j] = prev[j - 1] + 1;
            } else {
                curr[j] = Math.max(prev[j], curr[j - 1]);
            }
        }
        prev = curr;
    }
    return prev[n];
}

// Reconstruct the actual LCS string
function getLCS(text1, text2) {
    const m = text1.length, n = text2.length;
    const dp = Array.from({length: m + 1}, () => new Array(n + 1).fill(0));
    for (let i = 1; i <= m; i++)
        for (let j = 1; j <= n; j++)
            dp[i][j] = text1[i-1] === text2[j-1] ? dp[i-1][j-1]+1 : Math.max(dp[i-1][j], dp[i][j-1]);

    // Backtrack to find the LCS string
    let i = m, j = n, lcs = '';
    while (i > 0 && j > 0) {
        if (text1[i-1] === text2[j-1]) { lcs = text1[i-1] + lcs; i--; j--; }
        else if (dp[i-1][j] > dp[i][j-1]) i--;
        else j--;
    }
    return lcs;
}

⚠️ Common Mistakes

❌ Confusing LCS with Longest Common Substring

LCS allows gaps (subsequence). Longest Common Substring requires contiguous characters. For substring, reset to 0 when characters don't match.

❌ Off-by-one with 1-indexed DP table

dp is (m+1)×(n+1) with dp[0][*] and dp[*][0] as 0 (empty string). Access characters with text1[i-1] and text2[j-1].

💼 Interview Perspective

  • LCS is the foundation for Edit Distance, Shortest Common Supersequence, and Diff algorithms.
  • Edit Distance is LCS's cousin: same 2D DP table, different recurrence for non-matching characters.
  • Know how to reconstruct the actual LCS string by backtracking through the DP table.
  • Space optimization: use two 1D arrays or even one array with a temp variable.

📝 Quick Revision Notes

  • LCS: longest sequence present in both strings (not necessarily contiguous).
  • Recurrence: match → dp[i-1][j-1]+1, no match → max(dp[i-1][j], dp[i][j-1]).
  • Time O(mn), Space O(mn) or O(n) optimized.
  • Backtrack DP table to reconstruct actual LCS string.
  • Related: Edit Distance, Shortest Common Supersequence, diff tools.

🃏 Flash Cards

What is the LCS recurrence relation?
If text1[i-1] === text2[j-1]: dp[i][j] = dp[i-1][j-1] + 1 (extend LCS).
Else: dp[i][j] = max(dp[i-1][j], dp[i][j-1]) (skip one character from either string).
How is Edit Distance related to LCS?
Both use 2D DP on two strings. LCS finds the longest matching subsequence. Edit Distance finds the minimum operations (insert, delete, replace) to transform one string into another. editDistance = m + n - 2 * LCS (when only insert/delete allowed).
