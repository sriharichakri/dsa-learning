Frequency Array

Phase 2 Easy

📖 What is a Frequency Array?

A Frequency Array (also called a count array) tracks how many times each value appears in a dataset. It uses an array where the index represents the value and the stored number represents the count. It's the simplest form of hashing — mapping values to their frequencies.

🌍 Real-Life Analogy

Imagine a teacher counting how many students scored each grade. Instead of listing "A, B, A, C, A, B, A," she makes a tally chart: A = 4, B = 2, C = 1. That tally chart IS a frequency array! Each "bucket" (grade) holds a count of how many times it appeared.

⚙️ How It Works

JavaScript
// Basic frequency array for values 0..max
function buildFrequencyArray(arr) {
    const max = Math.max(...arr);
    const freq = new Array(max + 1).fill(0);
    
    for (let num of arr) {
        freq[num]++;
    }
    return freq;
}

// buildFrequencyArray([1, 3, 2, 1, 4, 1, 3, 2])
// → [0, 3, 2, 2, 1]  meaning: 0 appears 0 times, 1 appears 3 times, etc.

// Using a Map for any values (including negatives, strings)
function frequencyMap(arr) {
    const freq = new Map();
    for (let item of arr) {
        freq.set(item, (freq.get(item) || 0) + 1);
    }
    return freq;
}

// Character frequency for strings
function charFrequency(str) {
    const freq = new Array(26).fill(0);  // a-z
    for (let ch of str) {
        freq[ch.charCodeAt(0) - 97]++;
    }
    return freq;
}

Common Applications

JavaScript
// 1. Check if two strings are anagrams
function isAnagram(s, t) {
    if (s.length !== t.length) return false;
    const freq = new Array(26).fill(0);
    for (let i = 0; i < s.length; i++) {
        freq[s.charCodeAt(i) - 97]++;
        freq[t.charCodeAt(i) - 97]--;
    }
    return freq.every(count => count === 0);
}

// 2. Find the majority element (appears > n/2 times)
function majorityElement(nums) {
    const freq = new Map();
    for (let num of nums) {
        freq.set(num, (freq.get(num) || 0) + 1);
        if (freq.get(num) > nums.length / 2) return num;
    }
}

// 3. First non-repeating character
function firstUniqChar(s) {
    const freq = new Array(26).fill(0);
    for (let ch of s) freq[ch.charCodeAt(0) - 97]++;
    for (let i = 0; i < s.length; i++) {
        if (freq[s.charCodeAt(i) - 97] === 1) return i;
    }
    return -1;
}

// 4. Sort by frequency
function frequencySort(s) {
    const freq = new Map();
    for (let ch of s) freq.set(ch, (freq.get(ch) || 0) + 1);
    return [...s].sort((a, b) => freq.get(b) - freq.get(a) || a.localeCompare(b)).join('');
}

⚠️ Common Mistakes

❌ Using frequency array for large/negative values

Frequency arrays use the value as index. This only works for small non-negative integers. For large numbers, negatives, or strings, use a Map/Object instead.

❌ Array too small
// ❌ If max value is 100, array of size 100 misses index 100!
const freq = new Array(max).fill(0);  // Index 0..99

// ✅ Size should be max + 1
const freq = new Array(max + 1).fill(0);  // Index 0..100

💼 Interview Perspective

  • Frequency arrays are the foundation of counting sort and bucket sort.
  • For strings: use new Array(26).fill(0) — it's O(1) space and avoids hash overhead.
  • "Anagram" problems almost always need frequency comparison.
  • For sliding window problems, maintain a frequency map and update as the window moves.
  • Frequency array → prefix sum of frequency array = cumulative count (useful in counting sort).

📝 Quick Revision Notes

  • Frequency array: index = value, stored number = count. O(n) build, O(1) query.
  • For small non-negative integers: use array. For anything else: use Map.
  • Strings: 26-element array for lowercase letters. Character code - 97 = index.
  • Anagram check: build freq from both strings, compare (or increment/decrement with one array).

🃏 Flash Cards

When should you use a frequency array vs a hash map?
Frequency array: when values are small non-negative integers (0 to k). Fast, cache-friendly, no hash overhead. Hash map: when values are large, negative, strings, or sparse. More flexible but slower.
How do you check if two strings are anagrams using a frequency array?
Use one 26-element array. Increment for each character in string 1, decrement for each character in string 2. If all counts are 0 at the end → anagrams. O(n) time, O(1) space.
