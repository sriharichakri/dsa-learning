Loops & Iteration

Phase 1 Beginner Friendly

📖 What Are Loops?

A loop is a programming construct that repeats a block of code as long as a condition remains true. Instead of writing the same instruction 100 times, you write it once and tell the computer to repeat it.

Loops are the backbone of every algorithm — from searching through an array to processing millions of database records.

💡 Key Insight

Understanding loops is understanding time complexity. A single loop = O(n). A nested loop = O(n²). The number and depth of your loops directly determines how fast your code runs.

🌍 Real-Life Analogy

🏭 Assembly Line in a Factory

Imagine a car factory assembly line. Each car goes through the same steps: weld frame → paint → install engine → quality check. The factory doesn't write new instructions for each car — it loops the same process for every car until the batch is complete.

  • Clock ticking: Every second, the same action (tick) repeats. It stops when the alarm goes off (condition becomes false).
  • Brushing your teeth: You repeat the brushing motion ~200 times. Each stroke is one iteration.
  • Dealing cards: Deal one card to each player, repeat for the number of rounds.
  • Washing dishes: Pick dish → scrub → rinse → stack. Repeat until sink is empty.

⚙️ Types of Loops

1. For Loop — "I know how many times to repeat"

JavaScript
// Syntax: for (initialization; condition; update)
for (let i = 0; i < 5; i++) {
    console.log(i);  // 0, 1, 2, 3, 4
}

// Counting backwards
for (let i = 10; i >= 0; i--) {
    console.log(i);  // 10, 9, 8, ..., 0
}

// Iterating through an array
let fruits = ["apple", "banana", "cherry"];
for (let i = 0; i < fruits.length; i++) {
    console.log(fruits[i]);
}

2. While Loop — "Keep going until this changes"

JavaScript
// Syntax: while (condition) { body }
let count = 0;
while (count < 5) {
    console.log(count);  // 0, 1, 2, 3, 4
    count++;
}

// Unknown number of iterations
let n = 12345;
let digitCount = 0;
while (n > 0) {
    digitCount++;
    n = Math.floor(n / 10);
}
// digitCount = 5

3. Do-While Loop — "Run at least once, then check"

JavaScript
// Syntax: do { body } while (condition);
let input;
do {
    input = prompt("Enter a positive number:");
} while (input <= 0);
// Guarantees the prompt shows at least once

🔄 Nested Loops

A loop inside another loop. The inner loop runs completely for each iteration of the outer loop.

JavaScript
// Multiplication table (nested loop)
for (let i = 1; i <= 3; i++) {
    for (let j = 1; j <= 3; j++) {
        console.log(`${i} × ${j} = ${i * j}`);
    }
}
// 1×1=1, 1×2=2, 1×3=3, 2×1=2, 2×2=4, ... (9 total)
⚡ Nested Loop = O(n²) — Be Careful!

If outer loop runs n times and inner loop runs n times, total iterations = n × n = n². For n=10,000 that's 100,000,000 operations. Nested loops are often the first thing to optimize in interviews.

Classic Pattern: Print a Triangle

JavaScript
// Right triangle of stars
for (let i = 1; i <= 5; i++) {
    let row = "";
    for (let j = 0; j < i; j++) {
        row += "* ";
    }
    console.log(row);
}
// Output:
// *
// * *
// * * *
// * * * *
// * * * * *

🎮 Loop Control: break & continue

JavaScript
// BREAK — exit the loop immediately
for (let i = 0; i < 10; i++) {
    if (i === 5) break;   // Stop at 5
    console.log(i);       // 0, 1, 2, 3, 4
}

// CONTINUE — skip current iteration, go to next
for (let i = 0; i < 6; i++) {
    if (i === 3) continue; // Skip 3
    console.log(i);        // 0, 1, 2, 4, 5
}

// Real use: find first negative number
let arr = [4, 7, -2, 9, -5];
for (let i = 0; i < arr.length; i++) {
    if (arr[i] < 0) {
        console.log(`First negative: ${arr[i]} at index ${i}`);
        break;  // Found it, stop looking
    }
}

🔍 Dry Run: Sum of Digits

Let's trace through finding the sum of digits of 1234 using a while loop.

JavaScript
function sumOfDigits(n) {
    let sum = 0;
    while (n > 0) {
        sum += n % 10;     // Extract last digit
        n = Math.floor(n / 10); // Remove last digit
    }
    return sum;
}
sumOfDigits(1234); // → 10
1

n=1234, sum=0
2
1234 % 10 = 4 → sum = 0+4 = 4. n = 1234/10 = 123
n=123, sum=4
3
123 % 10 = 3 → sum = 4+3 = 7. n = 123/10 = 12
n=12, sum=7
4
12 % 10 = 2 → sum = 7+2 = 9. n = 12/10 = 1
n=1, sum=9
5
1 % 10 = 1 → sum = 9+1 = 10. n = 1/10 = 0. Loop ends (n=0).
sum=10 ✅ Answer!

⏱️ Complexity Analysis

Loop PatternTime ComplexityExample
Single loop (0 to n)O(n)Sum of array elements
Nested loop (n × n)O(n²)Bubble sort, brute force pair check
Triple nested loopO(n³)Matrix multiplication
Loop halving (n/2 each time)O(log n)Binary search
Two independent loopsO(n + m)Process two arrays separately

⚠️ Common Mistakes

❌ Infinite Loop (forgetting to update)
// ❌ i never changes — runs forever!
let i = 0;
while (i < 10) {
    console.log(i);
    // Missing i++!
}

// ✅ Always update the loop variable
while (i < 10) {
    console.log(i);
    i++;
}
❌ Off-by-One: < vs <=
let arr = [10, 20, 30]; // length = 3, indices: 0, 1, 2

// ❌ i <= arr.length goes to index 3 (undefined!)
for (let i = 0; i <= arr.length; i++) { ... }

// ✅ Use < length
for (let i = 0; i < arr.length; i++) { ... }
❌ Wrong loop type

Use for when you know the count. Use while when you don't. Use do-while when you need at least one execution. Choosing wrong makes code confusing.

💼 Interview Perspective

  • When you see a brute force with nested loops (O(n²)), immediately think: "Can I reduce this to O(n) using a hash map or two pointers?"
  • Know how to identify the time complexity by counting loop levels and iterations.
  • Pattern problems (stars, numbers) test your nested loop understanding — common in campus placements.
  • Be comfortable with while loops for digit manipulation (reverse number, palindrome check, count digits).
  • Understand for...of (values) vs for...in (keys) in JavaScript — interviewers test this.

📝 Quick Revision Notes

  • for = known iterations. while = unknown. do-while = at least once.
  • Single loop = O(n). Nested loop = O(n²). Halving loop = O(log n).
  • break exits the loop. continue skips current iteration.
  • Always ensure loop variable is updated to avoid infinite loops.
  • Array indices: 0 to length-1. Use < not <= with length.

🃏 Flash Cards

What is the time complexity of a nested loop (n × n)?
O(n²) — quadratic. For each of the n outer iterations, the inner loop runs n times. Total = n × n operations.
What is the difference between break and continue?
break exits the entire loop immediately. continue skips only the current iteration and proceeds to the next one.
When should you use a while loop vs a for loop?
Use for when iterations are known in advance (e.g., loop n times). Use while when iterations depend on a runtime condition (e.g., process until input is 0).
